Introducción

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML.

Permite a los desarrolladores web controlar el estilo y la apariencia de sus páginas web.

CSS es fundamental para separar la estructura de un documento HTML de su presentación visual.

En esta documentación, exploraremos la sintaxis, los selectores y las propiedades clave de CSS.

Sintaxis

La sintaxis básica de CSS consiste en un selector seguido de un bloque de declaraciones.

Una declaración incluye una propiedad y un valor, separados por dos puntos.

Ejemplo de sintaxis básica:

selector { propiedad: valor; }
  • Selector: Identifica el elemento HTML al que se aplicará el estilo.
  • Propiedad: Define el aspecto específico que se quiere cambiar.
  • Valor: Especifica cómo debe aplicarse esa propiedad.
Selectores

Los selectores en CSS permiten aplicar estilos a elementos HTML específicos.

Algunos de los selectores más comunes son:

  • Selector de elementos: Aplica estilos a todas las etiquetas de un tipo específico. Ejemplo: p { color: blue; }
  • Selector de clases: Aplica estilos a elementos con una clase específica. Ejemplo: .miClase { font-size: 20px; }
  • Selector de ID: Aplica estilos a un elemento con un ID específico. Ejemplo: #miID { margin: 10px; }
Propiedades

CSS tiene una amplia variedad de propiedades que controlan diferentes aspectos del diseño de una página web.

Algunas propiedades importantes incluyen:

  • color: Define el color del texto.body { color: #fff; }
  • background-color: Establece el color de fondo de un elemento.body { background-color: #f0f0f0; }
  • font-size: Controla el tamaño del texto.header { font-size: 1.5em; }
  • margin: Define el espacio exterior alrededor de un elemento.code { margin: 10px 0; }
  • padding: Establece el espacio interior dentro de un elemento.code { padding: 10px; }
Conclusión

CSS es una herramienta poderosa para dar estilo a las páginas web, permitiendo un control detallado sobre su apariencia.

Dominar la sintaxis, los selectores y las propiedades clave es esencial para cualquier desarrollador web.
